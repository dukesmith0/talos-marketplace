# /loci-prep — Meeting/Interview Prep

## Step 0: Resolve Environment
Run `loci vault` via Bash to get vault path.

## Steps
1. Identify what to prep for (interview, meeting, call)
2. Search `$VAULT/career/contacts/` and `$VAULT/career/applications/` for related people/companies
3. Read relevant vault results for context
4. Build structured prep doc:
   - **Background**: company, role, people involved
   - **Key points**: what to communicate
   - **Questions to ask**: prepared questions
   - **Potential challenges**: likely hard questions
   - **Quick reference**: stats, dates, names
5. Save to vault. Log: `loci log "prep: <brief outcome>"`

## Output
Concise prep document ready to skim before the meeting.
