# /loci-help — LOCI Command Reference

Display this reference when asked. If the user seems new, show the Quick Start section first.

## Quick Start by Role

**Notetaker / Knowledge Worker:**
`/loci-add` (capture anything) → `/loci-query` (find it later) → `/loci-reflect` (spot patterns)

**Developer:**
`/loci-init` (scaffold project) → `/loci-plan` (break down task) → `/loci-go` (execute) → `/loci-review` (verify)

**Student:**
`/loci-notes` (capture lectures) → `/loci-study` (Socratic tutoring) → `/loci-teach` (test understanding) → `/loci-learn` (quick lookup)

**Strategist / Marketer:**
`/loci-brainstorm` (SCAMPER ideas) → `/loci-ideate` (score & evaluate) → `/loci-research` (deep dive) → `/loci-think` (reason through)

**Job Seeker:**
`/loci-apply` (score fit) → `/loci-contact` (track people) → `/loci-prep` (interview prep) → `/loci-followup` (stay on top)

**Researcher:**
`/loci-add` (ingest papers) → `/loci-query` (cross-reference) → `/loci-research` (vault + web) → `/loci-hub` (explore topic graph)

**Project Manager:**
`/loci-plan` (decompose work) → `/loci-go` (track execution) → `/loci-ralph` (iterate until done) → `/loci-wrapup` (close session)

**Writer / Creator:**
`/loci-brainstorm` (generate) → `/loci-add` (capture drafts) → `/loci-query` (pull references) → `/loci-reflect` (find themes)

## All Commands

### Daily
| Command | What it does |
|---------|-------------|
| `/loci-query` | Search vault + project. Auto-depth, hub-aware. |
| `/loci-add` | Store knowledge. Auto-detects: text, document, extraction. |
| `/loci-log` | Timestamped entry to daily note. |

### Session
| `/loci-morning` | Briefing: calendar, priorities, open threads. |
| `/loci-wrapup` | Session close: update .loci, reindex, sync. |
