# /talos-help — TALOS Command Reference

Display this reference when asked.

## Daily
| Command | What it does |
|---------|-------------|
| `/talos-query` | Search vault + project for answers. Auto-depth. |
| `/talos-add` | Store knowledge. Auto-detects format. `--pin`/`--unpin` for working memory. |
| `/talos-log` | Append timestamped entry to daily note. |

## Session
| Command | What it does |
|---------|-------------|
| `/talos-morning` | Briefing: calendar, priorities, open threads, emails. |
| `/talos-wrapup` | Session close: update .talos, reindex, sync. |

## Weekly
| Command | What it does |
|---------|-------------|
| `/talos-reflect` | Pattern analysis on vault activity and gaps. |
| `/talos-maintain` | Diagnose → fix → clean → report. |

## Knowledge Graph
| Command | What it does |
|---------|-------------|
| `/talos-hub <tag>` | Deep-dive a tag hub: quality metrics, connections, gaps. |
| `/talos-tag <name>` | Create or lookup a tag hub page. |

## Anytime
| Command | What it does |
|---------|-------------|
| `/talos-sync` | Git sync vault. `--status` for brain report. |
| `/talos-help` | This reference. |

| `/talos-plan` | Task → checkable plan. |
| `/talos-go` | Execute next plan item. |
| `/talos-ralph` | Iterative loop with stuck detection. |
| `/talos-oneshot` | Quick task, no plan. |
| `/talos-review` | Code review. |
| `/talos-simplify` | Refactor for clarity. |
| `/talos-diff` | Git history by topic. |
| `/talos-risks` | Security/architecture risks. |
| `/talos-consolidate` | Compress .talos/ files. |
